New Tab

Open record in new tab.

Click here for the video tutorial

Key Features:

  • This Module will add a New Button in all tree view.
  • The moment you click on that button it will open the record that you have clicked in New Tab.
  • Record could be anything like Invoice details, Product details, Sale Order, Purchase Order etc.
  • This Module will help you to compare different record at same time.
